# Potato Leaf Disease Classifier

Esta aplicación usa una red neuronal (FCNN) entrenada con imágenes de hojas de papa para detectar enfermedades como tizón temprano, tizón tardío o clasificar como hoja sana.

## ¿Cómo usar?
1. Ejecutar y crear el odelo
2. Abrir el link generado
3. Sube una imagen de una hoja.
4. La página web  mostrará el diagnóstico.
